# keymouse

README: [English](README.md) | 日本語

KeyMouse は、キーボードから画面上の任意位置をクリックする Windows の軽量トレイ
アプリです。カーソルのあるモニター全体を固定 `40 × 25`（1,000セル）のグリッドにし、
各セルへ `A S D F G T R E W Q` による三文字コード（`AAA`、`AAS`、`AAD` …）を行優先で固定表示します。

画面キャプチャ、OpenCV、要素検出、テレメトリ、ネットワーク通信は使用しません。

## 使い方

1. 左クリックは `Shift` を二連打します。右クリックは `Alt+R`、ダブルクリックは `Alt+D` です。
2. カーソルのあるモニターに1,000個のラベルが表示されます。
3. 目的のセルの三文字を入力すると、三打鍵目でそのセル中心をクリックします。

選択中のキー:

- `H/J/K/L`: グリッドを左/下/上/右へ4px移動し、途中入力を消去
- `Shift+H/J/K/L`: 16px移動し、途中入力を消去
- `Backspace`: 入力済みの文字を一つ戻す
- `Esc`: キャンセル

## ビルド


- ソースビルドには Go 1.22 以上
